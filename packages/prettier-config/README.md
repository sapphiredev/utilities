## Installation

```sh
yarn add -D @sapphire/prettier-config
```

---

## Usage

Add the Prettier config to your `package.json`:

```json
{
	"name": "my-project",
	"prettier": "@sapphire/prettier-config"
}
```

Or to `prettierrc.js`:

```js
module.exports = require('@sapphire/prettier-config');
```
